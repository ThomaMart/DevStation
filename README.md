# 🚀 DevStation

> Infrastructure as Code for my Linux development workstation.

DevStation is an automated setup for a complete Linux development environment focused on DevOps, Cloud and Embedded Linux.

---

## ✨ Features

- Debian 13
- Docker
- Terraform
- Ansible
- KVM / Libvirt
- Virt-Manager
- GitHub CLI
- VS Code
- kubectl
- Helm
- Ollama

---

## Installation

```bash
git clone git@github.com:ThomaMart/DevStation.git

cd DevStation

./install.sh
```
